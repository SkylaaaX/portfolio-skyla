Stage 1 — Association Grenade

PME - Paris

« Secteur de l'innovation humaine »

Missions

  • Créer une plateforme afin de permettre aux utilisateurs de se connecter et converser via une application tierce.
  • Mettre en avant les travaux de l'équipe de modélisation 3D et créer un monde dans le même thème.
  • Ajouter une intégration pour créer des scènes réalistes sur Unity en utilisant VR SDK.

Outils utilisés

Unity 2022 : moteur de création
VS Code : modification partielle de certains éléments de jeu
GitHub : ajout de code via GitHub et Git
HTML/CSS/JS : création d'un menu contextuel
Trello : suivi du projet
Blender : modification d'assets et textures
Photoshop / Medibang Paint Pro : création de height maps, UV maps et filtres

Compétences acquises

  • Compétence technique : intégration et modification d'assets 3D dans Unity (gestion des scènes, matériaux, scripts et interactions).
  • Compétence organisationnelle : planification et suivi des tâches du projet avec Trello et GitHub pour respecter les délais et coordonner les développements.
  • Compétence relationnelle : collaboration avec les modélisateurs 3D et les autres membres de l'équipe pour assurer la cohérence entre besoins techniques et ressources graphiques.
